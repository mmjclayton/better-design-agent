## Whitespace Functions

1. **Grouping** - Space between groups separates unrelated content. Space within groups creates association (Gestalt proximity).

2. **Hierarchy** - More whitespace around an element = more visual importance. Headlines with generous margins stand out.

3. **Readability** - Line spacing (leading) of 1.4-1.6x font size for body text. Paragraph spacing of 0.5-1em between blocks.

4. **Breathing room** - Dense interfaces cause cognitive overload. Every element competes for attention.

## Information Density Guidelines

| Context | Approach | Example |
|---------|----------|---------|
| Consumer app | Generous whitespace, focus on one task | Landing page, onboarding |
| Productivity tool | Moderate density, clear sections | Email client, project manager |
| Data-dense dashboard | Tight but systematic spacing | Analytics, trading platform |
| Content/editorial | Very generous, typography-led | Blog, documentation |

## Common Violations
- Equal spacing everywhere (no hierarchy through whitespace)
- Text touching container edges (no internal padding)
- List items too tightly packed (line height under 1.4)
- Cards with no margin between them
- Section breaks with insufficient spacing (less than 2x component spacing)
- Headers that are closer to the content above than below (should be closer to their own section)
