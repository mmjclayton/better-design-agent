## The Nine Gestalt Principles

1. **Proximity** - Elements close together are perceived as a group. Use consistent spacing to create visual groups: tight spacing (4-8px) within a group, wider spacing (16-24px) between groups.

2. **Similarity** - Elements that look alike (color, shape, size) are perceived as related. Use consistent styling for same-function elements (all buttons look like buttons, all links look like links).

3. **Closure** - The mind fills in missing parts to perceive complete shapes. Cards, containers, and bordered areas don't need full borders if enough visual cues exist.

4. **Continuity** - Elements arranged along a line or curve are perceived as related. Align elements to create visual flow. Consistent left alignment creates a strong reading line.

5. **Figure-ground** - People perceive objects (figure) as separate from their background (ground). Use contrast, shadows, or overlays to separate interactive elements from their background.

6. **Common region** - Elements within a shared boundary are perceived as a group. Cards, panels, and containers create visual grouping without requiring proximity.

7. **Symmetry** - Symmetrical elements are perceived as belonging together. Use symmetry in layouts for balance, but asymmetry for visual hierarchy and emphasis.

8. **Common fate** - Elements moving in the same direction are perceived as a group. Animations and transitions should move related elements together.

9. **Focal point** - Elements that stand out visually (size, color, contrast) attract attention first. Use a single focal point per section to guide the user's eye.

## When to apply
- Proximity and common region: form field grouping, card layouts, navigation sections
- Similarity: button hierarchies, link styling, icon families
- Figure-ground: modals, dropdowns, tooltips, floating action buttons
- Focal point: primary CTAs, error states, notifications
