## Core Design Principles

1. **Material is the metaphor** - Surfaces, edges, and shadows provide grounding. Material metaphor provides consistent spatial relationships.

2. **Bold, graphic, intentional** - Typography, grids, space, scale, colour, and imagery guide visual treatments.

3. **Motion provides meaning** - Motion is meaningful and appropriate, serving to focus attention and maintain continuity.

## M3 Colour System

- **Dynamic colour** from a source colour (seed) that generates a full tonal palette
- Five key colours: Primary, Secondary, Tertiary, Neutral, Neutral Variant
- Each key colour generates a tonal range (0-100 tone values)
- Surface containers at 5 elevation levels using tonal overlay

## M3 Typography Scale

| Role | Default Size | Weight | Use |
|------|-------------|--------|-----|
| Display Large | 57px | 400 | Hero text |
| Headline Large | 32px | 400 | Page headings |
| Title Large | 22px | 400 | Section headings |
| Body Large | 16px | 400 | Primary body text |
| Body Medium | 14px | 400 | Secondary text |
| Label Large | 14px | 500 | Buttons, tabs |
| Label Small | 11px | 500 | Captions, metadata |

## M3 Component Standards

- **Touch targets**: 48x48dp minimum
- **Icon size**: 24dp default, 20dp dense
- **Border radius**: Range from 0dp (none) to full (pill shape)
- **Elevation**: 5 levels (0, 1, 3, 6, 8, 12dp)

## When to apply
- Reference for component sizing, spacing, and proportion standards
- Colour system evaluation for tonal consistency
- Motion guidelines for transition timing and patterns
